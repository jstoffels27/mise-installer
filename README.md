# Mise

AI-powered menu → Square catalogue builder.

## Install

Download `Mise_Installer.command` and double-click it.

> First run: right-click → Open to bypass Gatekeeper.

## What it does

Extracts menu items from a source file (Word doc, PDF, image, or plain text) using Goose AI and fills a blank Square catalogue XLSX template — ready to upload to the Square Dashboard.

## Requirements

- macOS with Python 3 (built in)
- Goose CLI configured with your Databricks credentials
- openpyxl — installed automatically by the installer
